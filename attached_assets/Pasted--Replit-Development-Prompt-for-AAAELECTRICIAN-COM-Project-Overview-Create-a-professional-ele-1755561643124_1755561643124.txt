# Replit Development Prompt for AAAELECTRICIAN.COM

## Project Overview
Create a professional electrician website with 60+ service pages (30 residential + 30 commercial), optimized for SEO, local search, and Google AdSense integration. The website should be fast, mobile-responsive, and conversion-focused.

## Technical Requirements

### Tech Stack
- **Frontend**: React.js with Next.js for SSR/SSG
- **Styling**: Tailwind CSS for responsive design
- **SEO**: Next.js built-in SEO features + react-helmet-async
- **Forms**: React Hook Form with validation
- **Analytics**: Google Analytics 4 integration
- **AdSense**: Google AdSense responsive ad units
- **Schema**: JSON-LD structured data
- **Performance**: Image optimization, lazy loading, code splitting

### Dynamic Page Generation
```typescript
// Generate state pages
export async function generateStaticParams() {
  const statePaths = locations.map(location => ({
    state: location.state.toLowerCase().replace(' ', '-')
  }));
  
  const cityPaths = locations.flatMap(location =>
    location.cities.map(city => ({
      state: location.state.toLowerCase().replace(' ', '-'),
      city: city.slug
    }))
  );
  
  const servicePaths = locations.flatMap(location =>
    location.cities.flatMap(city =>
      services.map(service => ({
        state: location.state.toLowerCase().replace(' ', '-'),
        city: city.slug,
        service: service.slug
      }))
    )
  );
  
  return [...statePaths, ...cityPaths, ...servicePaths];
}
```

### Sitemap Generation for Scale
```typescript
// Generate massive sitemap with all location combinations
export const generateSitemap = () => {
  const urls = [];
  
  // State pages
  locations.forEach(location => {
    urls.push(`https://aaaelectrician.com/${location.state.toLowerCase().replace(' ', '-')}-electrician-services`);
  });
  
  // City pages  
  locations.forEach(location => {
    location.cities.forEach(city => {
      urls.push(`https://aaaelectrician.com/${location.state.toLowerCase().replace(' ', '-')}/${city.slug}-electrician`);
    });
  });
  
  // Service + City combinations (18,000+ pages)
  locations.forEach(location => {
    location.cities.forEach(city => {
      services.forEach(service => {
        urls.push(`https://aaaelectrician.com/${location.state.toLowerCase().replace(' ', '-')}/${city.slug}/${service.slug}`);
      });
    });
  });
  
  return urls;
};
```
